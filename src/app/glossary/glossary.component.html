Glossary

PrivacyControl of one's own information, including it's distribution and usage

Personal InformationInformation about an individual

Sensitive InformationInformation that needs to be kept confidential

Personally Identifiable Information (PII)Information that can be used to directly or indirectly identify an individual

Data ControllerAn entity that stores and processes information

Data BreachA security breach where data is released without authorization. Also known as a data leak

Third PartyAn entity which does not directly interact with the user but provides auxillary services to the main data controller
