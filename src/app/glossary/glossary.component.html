Glossary

Privacy

Control over the collection, distribution, and usage of information about oneself, whether this information is directly or indirectly collected

Personal Information

Information about an individual

Sensitive Information

Information that needs to be kept confidential

Personally Identifiable Information (PII)

Information that can be used to directly or indirectly identify an individual

Deanonymization

The process of determining a person's identity from anonymous data by cross-referencing multiple data sources.

Data Controller

An entity that stores and processes information

Data Breach

A security breach where data is released without authorization. Also known as a data leak

Third Party

An entity which does not directly interact with the user but provides auxillary services to the main data controller
